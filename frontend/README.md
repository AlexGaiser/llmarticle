# Frontend - LLM Article Project

## Description
React + TypeScript + Tailwind CSS frontend for the LLM Article Project.
Built with Vite.

## Running Locally

### Prerequisites
- Node.js (v20+)

### Steps
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:5173

## Deployment
1. Build the project:
   ```bash
   npm run build
   ```
2. Deploy the `dist/` folder to a static host (e.g., AWS S3, Vercel, Netlify).

## Testing
Run unit tests with:
```bash
npm test
```
